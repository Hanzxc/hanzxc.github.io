Lamport OTP

A standard-library Python implementation of the Lamport hash-chain one-time password scheme.

  • Builds a seed-based hash chain with SHA-256.
  • Discloses passwords in reverse order, verified against a moving server anchor.
  • Tests for replay rejection, forged passwords, out-of-order use, and chain exhaustion.

TOTP Simulator

A standard-library Python implementation of time-based one-time passwords per RFC 6238 — the scheme behind authenticator apps.

  • Generates codes from a base32 secret with SHA-1, SHA-256, or SHA-512.
  • Verified against official RFC 6238 test vectors.
  • CLI: secret generation, code display, drift-tolerant verification, key-URI export.
