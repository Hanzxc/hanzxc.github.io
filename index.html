Lamport OTP

A standard-library Python implementation of the Lamport hash-chain one-time password scheme.

  • Builds a seed-based hash chain with SHA-256.
  • Discloses passwords in reverse order and verifies against a moving server anchor.
  • Includes tests for replay rejection, forged passwords, out-of-order use, and chain exhaustion.

TOTP Simulator

A standard-library Python implementation of time-based one-time passwords per RFC 6238, the scheme behind authenticator apps.

  • Generates TOTP codes from a base32 secret with SHA-1, SHA-256, or SHA-512.
  • Verified against official RFC 6238 test vectors.
  • CLI supports secret generation, code display, verification with clock-drift tolerance, and key-URI export.
